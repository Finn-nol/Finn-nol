**Hieringer Finn**

## Projektübersicht  
Dieses Projekt untersucht **Studienmotivation**, **Lernmethoden** sowie mögliche Zusammenhänge zwischen beiden.  
Die Daten stammen aus einer von Teilnehmenden ausgefüllten Umfrage und werden anschließend dokumentiert und ausgewertet.

## Ordnerstruktur  
Im Ordner "Data" befindet sich die Datenerhebung (`hierifin_ADLS_datenerhebung_2025.xlsx`)
Im Ordner "Doc" befindet sich der Datenerhebungsbericht (`Datenerhebungsbericht_Studienmotivation.docx`)

## Dateien im Überblick

### **Data/**
Enthält alle erhobenen **Rohdaten** der Umfrage.  
Format: `.xlsx`

### **Doc/**
Dokumente zur **Datenerhebung**, **Methodik** und **Auswertung**.  
Format: `.docx`

### **README.md**
Diese Projektbeschreibung.

## Ziel des Projekts  
- Untersuchung der Motivation von Studierenden  
- Analyse bevorzugter Lernmethoden  
- Ableitung von Mustern, Zusammenhängen und Empfehlungen  
- Aufbereitung der Ergebnisse für wissenschaftliche Dokumentation
